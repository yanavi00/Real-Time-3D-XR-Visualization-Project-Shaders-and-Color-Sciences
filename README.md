# Real-Time-3D-XR-Visualization-Project-Shaders-and-Color-Sciences
A Three.js application that uses custom GLSL shaders (targeting WebGL) to visualise color information extracted from an image or a video stream. The six standard color spaces (RGB, HSV, CIEXYZ, CIExyY, CIELAB and CIELCH) were implemented + elevation maps + a Lambertian directional lighting model.
